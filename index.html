Feature

Train a New Model

During the training phase, the face recognizer will open and read numerous image files. Additionally, it must be able to identify the individuals shown in these images. To facilitate this process, you'll establish a directory (training/, validation/, output/) structure that imparts essential information to your program.

Validate the Model

Model validation is a method employed to assess the performance of the trained model using data it hasn't encountered during training, but the developer, is familiar with. By having access to the correct labels for each image in this new set of data, you can gauge how well your model performs when given new dataset.
